# Federal Playbook — Hub Home (Preview)

Static preview of the Federal Playbook hub homepage for Capital Wealth, built from a Claude Design handoff.

This is the **homepage only** — internal nav links (`/federal-benefits-playbook/tools/...`, `/learn/...`, etc.) are not yet implemented and will 404.

## Files

- `index.html` — the hub homepage (originally `hub-home.html`)
- `hub-home.css` — page-specific styles
- `chrome.css` + `chrome.js` — shared header / footer
- `tokens.css` — design tokens (palette, type, spacing)

## Local preview

```sh
npx serve .
```

Then open the printed URL.
